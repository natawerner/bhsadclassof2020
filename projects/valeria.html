The walls on textile — Valeria Bukharova

Look around yourself. What do you see? What surrounds you everyday? Do you notice that visual noise everywhere?

Advertisements have become the new visual noise. They appear in our field of vision so often that we began to perceive them as something indivisible with the urban environment. It is everywhere: pasted on the clean facade of a new building, formed a bizarre pattern with other similar paper ads... Let's admit paper ads stuck on top of everything so that you can not get to the original surface! Let's imagine for a while... What if ads were so powerful that it moved into textiles? What would the prints look like if our clothes became like the city wall? This project is about a beautiful set of patterns, inspired by a visual noise!

And as an addition there is a further working on a real time pattern generator is going on. It has several modes: from the structured to the chaotic. So that you can create your own visual noise riso pattern by yourself! There are some patterns created by the generator.
